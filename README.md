# Installation in a PHP project

Run command:

    composer require fgsl/swagger-docs

For getting Composer: https://getcomposer.org/download/

# How to use

See sample below: 

```bash
vendor/bin/fsd
```

# Build

```bash
vendor/bin/phing
```
